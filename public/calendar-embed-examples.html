Option 2: Basic Calendar Embed (iframe)

Replace YOUR_CALENDAR_ID with your actual Google Calendar ID (usually your email).

<iframe src="https://calendar.google.com/calendar/embed?src=YOUR_CALENDAR_ID%40gmail.com&ctz=America%2FPhoenix" style="border: 0" width="800" height="600" frameborder="0" scrolling="no"> </iframe>

Demo: To see this working, you need to replace YOUR_CALENDAR_ID with your actual calendar ID.

Steps to get your calendar ID:

  1. Go to Google Calendar settings
  2. Click on your calendar
  3. Find "Calendar ID" in the integration section
  4. It will look like: rank@townranker.com or a long string ending in @group.calendar.google.com

Option 4: Google Calendar API Integration

Advanced integration using the Google Calendar API with your existing credentials.

<!-- Include Google API --> <script src="https://apis.google.com/js/api.js"></script> <div id="calendar-container"></div> <script> // Your existing API credentials const CLIENT_ID = '410008002816-pja5c9estqiloavso5dinnc7ij2squ45.apps.googleusercontent.com'; const API_KEY = 'AIzaSyAN8TZL-pbR-VRbgvjwVnjLJf9-SOGJABo'; const CALENDAR_ID = 'rank@townranker.com'; // Replace with your calendar ID // Discovery doc for Calendar API const DISCOVERY_DOC = 'https://www.googleapis.com/discovery/v1/apis/calendar/v3/rest'; // Authorization scopes const SCOPES = 'https://www.googleapis.com/auth/calendar.readonly'; function initializeGapiClient() { gapi.client.init({ apiKey: API_KEY, discoveryDocs: [DISCOVERY_DOC], }).then(() => { // Load calendar events loadCalendarEvents(); }); } function loadCalendarEvents() { gapi.client.calendar.events.list({ 'calendarId': CALENDAR_ID, 'timeMin': (new Date()).toISOString(), 'showDeleted': false, 'singleEvents': true, 'maxResults': 10, 'orderBy': 'startTime' }).then(response => { const events = response.result.items; displayEvents(events); }); } function displayEvents(events) { const container = document.getElementById('calendar-container'); if (events.length > 0) { let html = '<h3>Upcoming Events</h3><ul>'; events.forEach(event => { const when = event.start.dateTime || event.start.date; html += `<li>${event.summary} - ${when}</li>`; }); html += '</ul>'; container.innerHTML = html; } else { container.innerHTML = '<p>No upcoming events found.</p>'; } } // Load the API client gapi.load('client', initializeGapiClient); </script>

üìù Implementation Instructions

  1. Get Your Calendar ID:
    • Go to Google Calendar
    • Click Settings ‚Üí Settings
    • Select your calendar
    • Scroll to "Integrate calendar"
    • Copy your Calendar ID
  2. Choose Your Embed Style:
    • Option 1-2: Simple, no authentication needed
    • Option 3: Customizable appearance
    • Option 4: Full API integration (most flexible)
    • Option 5: Mobile-friendly
  3. Customize Parameters:
    • bgcolor: Background color
    • color: Calendar color theme
    • showTitle: Show/hide calendar title
    • showPrint: Show/hide print button
    • showCalendars: Show/hide calendar list
    • showTz: Show/hide timezone
    • mode: WEEK, MONTH, or AGENDA
